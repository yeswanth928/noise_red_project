Noise Reduction Using Deep Learning

This application helps you to reduce noise in your audio file using deep learning.

We use a convolution neural network to reduce noise.

Noise Reduction

This app uses a CNN trained with over 6500 white noise files and has an accuracy of 98% to reduce the white noise in audio file

Training Your Own Model

You can also train using your data and the application automatically saves the model and you can use it in future

Extract Audio

You can extract Audio from video files and use the audio file to reduce noise. You can extract the audio in .ogg, .mp3, .wav formats

Trimming Audio

You can Trim your Audio files and use only that part where you want noise reduction as Deep learning operations often consume a significant amount of time

Generate Spectrograms

You can generate spectrograms for entire folder of audio files at a time instead of using one audio file at a time.And You can choose between Hz or Log type
